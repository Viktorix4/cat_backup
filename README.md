# Backup Cats

Скрипт для автоматического сохранения изображений котов с надписями с сервиса [Cataas](https://cataas.com).

## Файлы

- `backup_cats.py` — основной скрипт, скачивает изображение кота с заданной надписью и сохраняет его
- `backup_log.json` — файл лога, в который записываются данные о каждом сохранённом изображении (время, текст, имя файла)
